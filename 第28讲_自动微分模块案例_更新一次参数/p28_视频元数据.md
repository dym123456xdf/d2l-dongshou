## 知识点清单

### 核心概念
1. 只有标量才能求导
2. requires_grad=True 开启自动微分
3. backward() 自动计算梯度
4. W.data 获取参数值
5. W.grad 获取梯度值

### API/函数
| 函数名 | 用途 | 重要程度 |
|--------|------|----------|
| torch.tensor(..., requires_grad=True) | 创建可求导张量 | ⭐⭐⭐ |
| loss.backward() | 反向传播求梯度 | ⭐⭐⭐ |
| loss.sum().backward() | 保证标量求导 | ⭐⭐⭐ |
| tensor.grad | 获取梯度 | ⭐⭐⭐ |
| tensor.data | 获取参数值 | ⭐⭐⭐ |

### 代码片段

```python
import torch

# 1. 定义初始权重
W = torch.tensor(10.0, requires_grad=True, dtype=torch.float32)

# 2. 定义损失函数 (模拟)
loss = 2 * W ** 2  # loss = 2W²

# 3. 打印梯度函数类型
print(loss.grad_fn)  # <MulBackward0>

# 4. 计算梯度
loss.backward()

# 5. 获取梯度
print(W.grad)  # tensor(40.) 因为 d(loss)/dW = 4W = 4*10 = 40

# 6. 更新权重
learning_rate = 0.01
W.data = W.data - learning_rate * W.grad
# 10 - 0.01 * 40 = 9.6

print(f"更新后的权重: {W.data}")  # 9.6
```
