## 内容结构分析

### 核心概念
通过代码演示如何利用自动微分模块对参数进行一次更新

### 关键点：只有标量才能求导
- PyTorch自动微分只支持对**标量**求导
- 如果loss是向量，需要用 `.sum()` 或其他方法转为标量
- 标准写法: `loss.sum().backward()`

### 权重更新流程
1. 定义初始权重 W = 10（requires_grad=True）
2. 定义损失函数 loss = 2W²
3. 调用 `loss.backward()` 计算梯度
4. 获取梯度 W.grad
5. 根据公式更新: W = W - 学习率 × 梯度

### 代码演示
```python
import torch

# 1. 定义初始权重
W = torch.tensor(10.0, requires_grad=True, dtype=torch.float32)

# 2. 定义损失函数
loss = 2 * W ** 2  # loss = 2W²，求导后是4W

# 3. 计算梯度
loss.backward()

# 4. 获取梯度
print(W.grad)  # 输出: 40 (因为4*10=40)

# 5. 更新权重
W.data = W.data - 0.01 * W.grad  # 10 - 0.01*40 = 9.6
```

### 关于data属性
- `tensor.data` 获取张量的具体数值
- `tensor.grad` 获取梯度值
- 注意：date是日期，data是数据，不要混淆
