## 核心知识点

### 1. 索引类型总览

| 类型 | 说明 | 掌握程度 |
|------|------|----------|
| 简单行列索引 | `t[row, col]` | ★★★★★ 必须 |
| 列表索引 | `t[[0,1], [2,3]]` | ★★★☆☆ 了解 |
| 范围索引 | `t[0:3, 1:4]` | ★★★★★ 必须 |
| 布尔索引 | `t[t > 5]` | ★★★★☆ 必须 |
| 多维索引 | 按轴获取数据 | ★★★★★ 必须 |

### 2. 简单行列索引
```python
t = torch.tensor([[1, 2, 3],
                  [4, 5, 6]])
t[0, 1]   # 第0行第1列 = 2
t[1, :]   # 第1行所有列 = [4, 5, 6]
t[:, 2]   # 所有行第2列 = [3, 6]
```

### 3. 列表索引（高级）
```python
t[[0, 1], [0, 2]]  # 获取 (0,0) 和 (1,2) 位置
```

### 4. 范围索引（切片）
```python
t[0:3, 1:4]   # 前3行，1-3列（左闭右开）
t[:, :2]      # 所有行，前2列
t[1:, 2:]      # 第1行之后，第2列之后
```

### 5. 布尔索引
```python
t[t > 5]              # 返回满足条件的所有元素
t[2, t[2] > 5]        # 第2行中大于5的元素
```

### 6. 多维索引（重点）
```python
t = torch.randn(3, 4, 5)  # 3D张量

t[0]             # 零轴（第0个）上的第一个数据
t[0, 0]           # 一轴（第1个）上的第一个数据
t[0, 0, 0]        # 二轴（第2个）上的第一个数据（具体元素）
```

## 轴的顺序
- **零轴 (dim=0)**: 最外层
- **一轴 (dim=1)**: 第二层
- **二轴 (dim=2)**: 最内层

## 需掌握重点

2. **范围索引**：最常用的切片操作
