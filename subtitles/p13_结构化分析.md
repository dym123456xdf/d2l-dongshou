## 内容概要
本讲主要介绍PyTorch中创建张量的多种方法，包括：
1. 指定类型创建（int/float/Tensor）
2. 从NumPy数组创建
3. 线性与随机张量创建
4. 全零/全一/指定值张量创建

## 核心知识点

### 1. 指定类型张量创建
| API | 说明 |
|-----|------|
| `torch.randint()` | 整数类型（注意与int的区别）|
| `torch.float` / `torch.float32` | 浮点类型 |
| `torch.int` / `torch.int64` | 整数类型 |
| `torch.Tensor` | 默认类型（通常是float）|

### 2. 从现有数据创建
| API | 说明 |
|-----|------|
| `torch.from_numpy(ndarray)` | 从NumPy数组创建 |
| `torch.tensor(data)` | 从数据直接创建（推荐）|

### 3. 线性空间张量
| API | 说明 |
|-----|------|
| `torch.arange(start, end, step)` | 步长等差，遵循左闭右开 [start, end) |
| `torch.linspace(start, end, steps)` | 固定步数，均匀分布 |

### 4. 随机张量
| API | 说明 |
|-----|------|
| `torch.rand()` | [0,1) 均匀分布 |
| `torch.randn()` | 均值0标准差1的正态分布 |
| `torch.randint(low, high, shape)` | 指定范围整数 |

### 5. 特殊值张量
| API | 说明 |
|-----|------|
| `torch.zeros(shape)` | 全零张量 |
| `torch.ones(shape)` | 全一张量 |
| `torch.full(shape, value)` | 指定值填充 |

### 6. 随机种子
| API | 说明 |
|-----|------|
| `torch.manual_seed(seed)` | 手动设置随机种子 |

### 第三部分：线性与随机张量 (8:00-20:00)
- **Arange vs Linspace**: 
  - arange: 步长固定，范围左闭右开
  - linspace: 数量固定，范围左右都闭
- **随机种子**: manual_seed 的作用
- **三种随机分布**:
  - rand: 均匀分布 [0,1)
  - randn: 标准正态分布 (μ=0, σ=1)
  - randint: 整数均匀分布 [low, high)
