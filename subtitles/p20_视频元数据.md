## 轴的顺序
| 轴 | 层级 | 说明 |
|----|------|------|
| 零轴(dim=0) | 最外层 | 索引第0个 |
| 一轴(dim=1) | 第二层 | 索引第1层 |
| 二轴(dim=2) | 最内层 | 索引元素 |

## API清单

### 索引相关
| API | 说明 |
|-----|------|
| `t[dim0]` | 零轴索引 |
| `t[dim0, dim1]` | 零轴+一轴 |
| `t[dim0, dim1, dim2]` | 具体元素 |

### 变形相关
| API | 说明 |
|-----|------|
| `t.view(shape)` | 改变形状（共享内存）|
| `t.reshape(shape)` | 改变形状（可能复制）|
| `t.transpose(dim1, dim2)` | 交换轴 |
| `t.permute(dims)` | 任意轴排列 |

## 代码示例
```python
# 三维张量
t = torch.randn(2, 3, 4)
print(t.shape)  # torch.Size([2, 3, 4])

# 多维索引
print(t[0].shape)     # torch.Size([3, 4])
print(t[0, 0].shape)  # torch.Size([4])
print(t[0, 0, 0])     # tensor(0.1234)

# 变形
t_view = t.view(12, 2)      # (2*3*4, 2)
t_reshape = t.reshape(4, 6) # (4, 6)

# 转置
t_t = t.transpose(0, 2)  # (4, 3, 2)
```
