## 内容概要
本讲主要介绍张量与NumPy之间的相互转换，以及标量张量的值提取：
1. 张量转NumPy数组
2. NumPy数组转张量

## 核心知识点

### 1. 张量与NumPy转换

| 转换方向 | API | 内存共享 | 说明 |
|----------|-----|----------|------|
| Tensor → NumPy | `tensor.numpy()` | 共享内存（浅拷贝）| |
| Tensor → NumPy | `tensor.numpy().copy()` | 不共享（深拷贝）| |
| NumPy → Tensor | `torch.from_numpy(ndarray)` | 共享内存（浅拷贝）| |
| NumPy → Tensor | `torch.tensor(ndarray)` | 不共享 | **推荐使用** |

### 2. 浅拷贝 vs 深拷贝
| 类型 | 行为 |
|------|------|
| 浅拷贝 | 只拷贝第一层，后续共享 |
| 深拷贝 | 所有层都拷贝，独立一份 |

### 3. 标量张量
- 定义：只有一个元素的张量
- 值提取：`tensor.item()` - **必须背下来**
- 返回Python标量值

## 场景代码示例

### 场景一：张量 → NumPy
```python
import torch
import numpy as np

tensor = torch.tensor([1, 2, 3])
# 浅拷贝（共享内存）
nd_array = tensor.numpy()
# 深拷贝（不共享内存）
nd_array_copy = tensor.numpy().copy()
```

### 场景二：NumPy → 张量
```python
nd_array = np.array([1, 2, 3])
# 共享内存方式
tensor1 = torch.from_numpy(nd_array)
# 不共享内存（推荐）
tensor2 = torch.tensor(nd_array)
```

### 场景三：标量张量提取值
```python
scalar_tensor = torch.tensor(30)
value = scalar_tensor.item()  # 返回 30
```

## 共享内存机制图解

```
张量 ──numpy()──→ NumPy数组（共享内存）
   │                    │
   │                    ▼
   │              修改一方的值
   │              会影响另一方
   │
   └───numpy().copy()──→ NumPy数组（独立内存）
```
