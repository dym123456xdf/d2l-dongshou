## API清单

### 运算API
| API | 符号 | 说明 |
|-----|------|------|
| `torch.add(a, b)` | `+` | 加法 |
| `torch.sub(a, b)` | `-` | 减法 |
| `torch.multiply(a, b)` | `*` | 乘法 |
| `torch.div(a, b)` | `/` | 除法 |
| `torch.neg(a)` | 无 | 取反 |

### In-place版本
| API | 说明 |
|-----|------|
| `a.add_(b)` | 加到a上 |
| `a.sub_(b)` | 从a减去b |
| `a.mul_(b)` | 乘到a上 |
| `a.div_(b)` | 除以b |
| `a.neg_()` | a变为相反数 |

## 代码示例
```python
import torch

t = torch.tensor([1, 2, 3])

# 符号运算
print(t + 10)    # [11, 12, 13]
print(t - 1)     # [0, 1, 2]
print(t * 2)     # [2, 4, 6]
print(t / 2)     # [0.5, 1.0, 1.5]
print(-t)        # [-1, -2, -3]

# In-place
t.add_(10)       # t变为[11, 12, 13]，原数据被修改
```
