## 核心知识点

### 1. 聚合运算（有dim参数）

| API | 说明 | dim=0 | dim=1 |
|-----|------|-------|-------|
| `torch.sum()` | 求和 | 按列求和 | 按行求和 |
| `torch.mean()` | 均值 | 按列均值 | 按行均值 |
| `torch.max()` | 最大值 | 每列最大值 | 每行最大值 |
| `torch.min()` | 最小值 | 每列最小值 | 每行最小值 |

**dim参数记忆**：
- `dim=0` → 跨行操作 → 得到列统计 → **零列**
- `dim=1` → 跨列操作 → 得到行统计 → **一行**

### 2. 幂运算

| API | 说明 | 示例 |
|-----|------|------|
| `torch.pow(a, n)` | n次幂 | `torch.pow(t, 2)` |
| `t ** n` | n次幂（**符号**）| `t ** 2` |
| `torch.sqrt(t)` | 平方根 | |
| `t ** 0.5` | 开平方（用符号）| |

### 3. 指数与对数

| API | 说明 |
|-----|------|
| `torch.exp(t)` | e的t次方 |
| `torch.log(t)` | 以e为底的对数 |
| `torch.log2(t)` | 以2为底的对数 |
| `torch.log10(t)` | 以10为底的对数 |

## 重要公式
```
e^x = exp(x)
log_e(x) = ln(x) = log(x)
log_2(x) = log2(x)
log_10(x) = log10(x)
```

## dim参数详解
```python
t = torch.tensor([[1, 2, 3],
                  [4, 5, 6]])

torch.sum(t, dim=0)  # tensor([5, 7, 9])  - 每列求和
torch.sum(t, dim=1)  # tensor([ 6, 15])  - 每行求和
```

## 需不需要背？
| 函数 | 态度 |
|------|------|
| sum/max/mean | 必须会用，符号和函数都要 |
| min | 后边案例会用到 |
| power/sqrt | 用 `**` 符号更简单，不用背API |
| exp/log系列 | 了解即可，符号更常用 |

## 教学重点
1. **dim参数是重点**：dim=0是列，dim=1是行
2. **符号比API更好记**：`**` 代替pow，`t.sqrt()` 代替torch.sqrt()
3. 对数三种底数：e, 2, 10
