## 核心知识点

### 1. 自动微分 (Auto-grad) 原理
```
输入X → [X·W + B] → Z(预测值) → Loss(损失函数) → 求导 → 反向传播 → 更新W和B
```

### 2. 核心概念对应

| 符号 | 含义 | 说明 |
|------|------|------|
| X | 输入特征 | 一般表示X1（特征） |
| W | 权重 | Weight |
| B | 偏置 | Bias |
| Z | 预测值 | 相当于Y，用Z标记与图一致 |
| Y | 真实值 | Ground Truth |
| Loss | 损失函数 | 预测与真实的差距 |
| ∇ | 梯度 | 偏导数 |

### 3. 传播方向
| 方向 | 名称 | 过程 |
|------|------|------|
| 从左到右 | 前向传播 | X → Z（计算预测值）|
| 从右到左 | 反向传播 | Loss → 更新W和B |

### 4. 多元线性回归公式
```
Y = X·W + B
```
或写为：
```
Z = X·W + B
```

### 5. 自动微分的任务
- 计算梯度（求导）
- 反向传播更新参数
