## 内容概要
本讲介绍两种重要的张量乘法运算：
1. **点乘**（对应元素相乘 / Hadamard积）
2. **矩阵乘法**（matmul / @ 符号 / mm）

## 核心知识点

### 1. 点乘 (Element-wise Multiplication)
**定义**：对应位置元素相乘

```
[1,2,3,4] ⊙ [5,6,7,8] = [1×5, 2×6, 3×7, 4×8] = [5, 12, 21, 32]
```

**条件**：两个张量形状完全相同

| API | 说明 |
|-----|------|
| `a * b` | **推荐** 符号写法 |
| `torch.multiply(a, b)` | 函数写法 |
| `a.mul(b)` | 方法写法 |

### 2. 矩阵乘法 (Matrix Multiplication)
**条件**：A的列数 = B的行数
**结果**：A的行数 × B的列数

```
A (m×n) @ B (n×p) = C (m×p)
```

| API | 说明 |
|-----|------|
| `a @ b` | **推荐** 符号写法 |
| `torch.matmul(a, b)` | 通用函数（支持多维广播）|
| `a.matmul(b)` | 方法写法 |
| `torch.mm(a, b)` | 仅限2D矩阵 |
| `torch.dot(a, b)` | 来自NumPy的一维向量点积 |

### 3. 广播与维度规则
- 矩阵乘法要求**最后两个维度**满足条件
- 高维张量：只取最后两维做矩阵乘法
- 例如：5维张量 → 第4维和第5维满足 A列=B行

## 示例代码

### 点乘示例
```python
t1 = torch.tensor([1, 2, 3, 4])
t2 = torch.tensor([5, 6, 7, 8])
result = t1 * t2  # [5, 12, 21, 32]
```

### 矩阵乘法示例
```python
# 2D矩阵
A = torch.randn(2, 3)  # 2行3列
B = torch.randn(3, 4)  # 3行4列
C = A @ B              # 2行4列

# 多维广播
# 最后两维做矩阵乘法
```

## 易混点：点乘 vs 矩阵乘法
| 特征 | 点乘 | 矩阵乘法 |
|------|------|----------|
| 符号 | `*` | `@` |
| 条件 | 形状完全相同 | A列=B行 |
| 结果形状 | 与输入相同 | A行×B列 |
| 用途 | 元素级运算 | 线性变换 |

## 教学重点
1. **点乘用 `*`**，**矩阵乘法用 `@`**
2. 矩阵乘法条件必须记住：A列=B行
3. 多维张量只关心最后两维
