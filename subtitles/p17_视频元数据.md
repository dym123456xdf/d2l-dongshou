## 涵盖主题
1. **点乘运算**
   - 对应元素相乘
   - `*` 符号
   - `torch.multiply()`

2. **矩阵乘法**
   - 条件：A列=B行
   - `@` 符号
   - `torch.matmul()`
   - `torch.mm()` - 2D专用

3. **广播机制**
   - 多维张量的最后两维
   - 高维矩阵乘法规则

## API清单

| API | 类型 | 说明 |
|-----|------|------|
| `a * b` | 符号 | 点乘（元素级）|
| `a @ b` | 符号 | 矩阵乘法（**推荐**）|
| `torch.multiply(a, b)` | 函数 | 点乘 |
| `torch.matmul(a, b)` | 函数 | 矩阵乘法（通用）|
| `torch.mm(a, b)` | 函数 | 仅2D矩阵 |
| `torch.dot(a, b)` | 函数 | NumPy风格的点积 |

## 示例

### 点乘
```python
t1 = torch.tensor([1, 2, 3, 4])
t2 = torch.tensor([5, 6, 7, 8])
t1 * t2  # tensor([ 5, 12, 21, 32])
```

### 矩阵乘法
```python
A = torch.randn(2, 3)
B = torch.randn(3, 4)
A @ B   # shape: (2, 4)
```
