## 涵盖主题
1. **简单行列索引**
   - 按位置获取元素
   - 按行/列获取

2. **列表索引**
   - 同时获取多个位置

3. **范围索引（切片）**
   - `start:stop` 格式
   - 左闭右开

4. **布尔索引**
   - 条件筛选

5. **多维索引**
   - 按轴逐层深入
   - 核心难点

## API清单

| API | 说明 |
|-----|------|
| `t[row, col]` | 简单行列索引 |
| `t[[r1,r2], [c1,c2]]` | 列表索引 |
| `t[r1:r2, c1:c2]` | 范围索引 |
| `t[condition]` | 布尔索引 |
| `t[dim0, dim1, dim2]` | 多维索引 |

## 多维索引轴顺序
```
零轴(dim=0) → 最外层
一轴(dim=1) → 第二层
二轴(dim=2) → 最内层
```

## 代码示例
```python
torch.manual_seed(2)
t = torch.randint(0, 10, (5, 5))

# 简单索引
t[0, 1]    # 第0行第1列

# 列表索引
t[[0,1], [0,2]]  # (0,0)和(1,2)

# 范围索引
t[0:3, 1:4]  # 前3行，1-4列

# 布尔索引
t[t > 5]     # 大于5的所有元素

# 多维索引（3D张量）
t = torch.randn(3, 4, 5)
t[0]         # 第0个矩阵
t[0, 0]       # 第0个矩阵的第0行
t[0, 0, 0]    # 具体元素值
```
