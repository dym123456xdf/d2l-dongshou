## API清单

### 聚合运算（有dim参数）
| API | 说明 |
|-----|------|
| `torch.sum(t, dim)` | 求和 |
| `torch.mean(t, dim)` | 均值 |
| `torch.max(t, dim)` | 最大值 |
| `torch.min(t, dim)` | 最小值 |

### 幂运算
| API | 说明 |
|-----|------|
| `torch.pow(t, n)` | n次幂 |
| `t ** n` | **推荐符号** |
| `torch.sqrt(t)` | 平方根 |
| `t ** 0.5` | 开平方 |

### 指数对数
| API | 说明 |
|-----|------|
| `torch.exp(t)` | e^t |
| `torch.log(t)` | ln(t) |
| `torch.log2(t)` | log_2(t) |
| `torch.log10(t)` | log_10(t) |

## dim参数记忆法
| dim值 | 操作方向 | 结果形状 | 口诀 |
|-------|----------|----------|------|
| dim=0 | 跨行（向下）| 每列统计 | 零是列 |
| dim=1 | 跨列（向右）| 每行统计 | 一是行 |

## 代码示例
```python
t = torch.tensor([[1., 2., 3.],
                  [4., 5., 6.]])

# 聚合运算
torch.sum(t, dim=0)   # tensor([5., 7., 9.])
torch.sum(t, dim=1)   # tensor([ 6., 15.])

# 幂运算
t ** 2                # 平方
t ** 0.5               # 开平方

# 指数对数
torch.exp(t)          # e^t
torch.log(t)          # ln(t)
```
