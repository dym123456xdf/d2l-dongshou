## 核心知识点

### 1. 三维张量的理解
```python
t = torch.randn(2, 3, 4)  # 2个3行4列的矩阵
```
- 零轴：2个矩阵
- 一轴：每个矩阵3行
- 二轴：每行4列

### 2. 多维索引详解

| 索引 | 含义 | 结果形状 |
|------|------|----------|
| `t[0]` | 零轴第0个（第一个矩阵）| (3, 4) |
| `t[0, 0]` | 一轴第0个（第一行）| (4,) |
| `t[0, 0, 0]` | 具体元素 | scalar |

### 3. 数据变形操作

| API | 说明 | 特点 |
|-----|------|------|
| `t.view(shape)` | 改变形状 | 共享内存，需连续 |
| `t.reshape(shape)` | 改变形状 | 可能复制数据 |
| `t.transpose(dim1, dim2)` | 交换两个轴 | |
| `t.permute(dims)` | 任意轴排列 | |

### 4. 轴的深入理解
- **零轴**: 最外层，索引第0个元素
- **一轴**: 第二层
- **二轴**: 最内层
- **倒序**: 最后一个轴

## 多维索引示例
```python
t = torch.randn(2, 3, 4)

# 零轴索引
print(t[0].shape)    # torch.Size([3, 4])

# 一轴索引  
print(t[0, 0].shape)  # torch.Size([4])

# 二轴索引
print(t[0, 0, 0])     # scalar
```

## view vs reshape
| 操作 | 内存 | 连续性要求 |
|------|------|------------|
| view | 共享 | 必须连续 |
| reshape | 可能复制 | 无要求 |
