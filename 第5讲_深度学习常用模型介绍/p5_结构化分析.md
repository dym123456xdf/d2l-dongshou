## 内容概要
本讲主要介绍常见的深度学习模型及其应用场景：
1. 卷积神经网络(CNN) - 图像处理
2. 循环神经网络(RNN) - 文本/语音处理
3. 自编码器 - 降维
4. 生成对抗网络(GAN) - 图像/视频生成
5. Transformer - 自然语言处理

## 核心知识点

### 1. 卷积神经网络 (CNN)
- **应用**: 图像处理
- **层次结构**: 卷积层 + 池化层 + 全连接层
- **任务**: 图像分类、目标检测、图像分割
- **案例**: CIFAR-10图像分类

### 2. 循环神经网络 (RNN)
- **应用**: 文本、语音等序列数据
- **特点**: 后边数据对前边数据有依赖
- **任务**: 文本生成、机器翻译、情感分析

### 3. 自编码器
- **类型**: 无监督学习
- **应用**: 降维

### 4. 生成对抗网络 (GAN)
- **应用**: 图像生成、视频合成
- **早期应用较多，现在相对少**

### 5. Transformer
- **应用**: 自然语言处理
- **特点**: 摒弃传统RNN，采用自注意力机制
- **地位**: 目前主流，是LLM的基础

### 神经网络层次
```
输入层 → 隐藏层(卷积层/池化层/全连接层) → 输出层
```

## 模型与应用场景对照表
| 模型 | 应用领域 | 具体任务 |
|------|----------|----------|
| CNN | 图像 | 分类、目标检测、分割 |
| RNN | 文本/语音 | 生成、翻译、情感分析 |
| 自编码器 | 降维 | 数据压缩 |
| GAN | 图像/视频 | 生成任务 |
| Transformer | NLP | 机器翻译、文本生成 |

## 深度学习主要处理对象
1. **图像** → CNN
2. **文本** → RNN/Transformer

## 教学要点
1. CNN处理图像，RNN处理文本
2. Transformer是LLM的基础，后边会重点讲
3. 注意区分：TensorFlow(旧框架) vs Transformer(新架构)
