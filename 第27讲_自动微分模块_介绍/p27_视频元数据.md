## 知识点清单

### 核心概念
1. 自动微分模块 = 求导工具
2. 反向传播(BP) = 基于梯度更新参数
3. 权重更新公式: W新 = W旧 - 学习率 × 梯度
4. requires_grad - 开启自动微分
5. backward - 自动计算梯度

### API/函数
| 函数名 | 用途 | 重要程度 |
|--------|------|----------|
| tensor.requires_grad=True | 开启自动微分 | ⭐⭐⭐ |
| loss.backward() | 反向传播，求导 | ⭐⭐⭐ |
| tensor.grad | 获取梯度值 | ⭐⭐⭐ |
| tensor.data | 获取参数值 | ⭐⭐⭐ |

### 关键公式
```
线性模型: Y = WX + B
损失函数(MSE): loss = (预测值 - 真实值)²
权重更新: W新 = W旧 - 学习率 × 梯度
梯度 = 损失函数的导数
```
