## 内容概要
本讲主要介绍深度学习的四个核心特点：
1. 多层非线性变换
2. 自动提取特征
3. 大数据和计算能力
4. 可解释性差

## 核心知识点

### 1. 多层非线性变换
- 深度学习由多个层次组成
- 每层都应用非线性激活函数
- 每个神经元先做加权求和，再做激活函数
- 目的：使框架既能处理回归也能处理分类

### 2. 自动提取特征
- 由算法自动帮助提取特征
- 每个神经元都与前层所有神经元连接（全连接）
- 无需人工特征工程

### 3. 大数据和计算能力
- 深度学习通常需要大量标注数据
- 需要强大的计算资源（GPU）
- 硬件举例：A100约15万、H100约20万
- 通过device参数切换CPU/GPU：device='cuda:0' 或 device='cpu'

### 4. 可解释性差（黑箱问题）
- 模型内部运作机制相对不透明
- 只能看到结果，看不到过程
- 只能凭经验去测（如神经元数量、层数）
- 开发经验：浅层神经元多，往后逐层递减

## 公式与符号

### 神经元计算流程
```
加权求和 → 激活函数 → 输出
```

### 权重更新公式
```
W新 = W旧 - 学习率 × 梯度
```
