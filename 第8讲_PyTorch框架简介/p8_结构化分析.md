## 核心知识点

### 1. PyTorch定义
- 基于Python语言的深度学习框架
- 将数据封装成张量(Tensor)进行处理
- 广泛应用于学术研究和工业界
- 底层由C/C++编写

### 2. PyTorch安装方式
**方式一：Anaconda终端安装**
```bash
pip install torch
# 国内镜像
pip install torch -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**方式二：PyCharm终端安装**
- 打开Terminal终端
- 执行pip install torch

**方式三：PyCharm界面安装**
- File → Settings → Project → Python Interpreter
- 点击+号搜索torch
- 点刷新选1store安装

### 3. 核心特性

#### 自动微分 (Auto-grad)
- 本质：**求导**
- 结合反向传播(BP)来更新参数
- 公式：W新 = W旧 - 学习率 × 梯度

#### 动态计算图
- 边计算边生成，可以边做边改
- 区别于TensorFlow的静态图（先定义再计算）

#### GPU加速
- 通过device参数切换：device='cuda:0' (GPU) / device='cpu' (CPU)

### 4. 发展史与市场竞争
```
Torch(C/C++, 高校用, 无GPU) 
  ↓ 简化API，加GPU支持
→ PyTorch(Facebook, 动态图, API简单)
  ↓
→ TensorFlow(Google, 静态图, API复杂, 市场流失)
  ↓ 2.x改进
→ TensorFlow2.x(Keras集成, 动态图支持)
```

### 5. PyTorch优势
- 动态图计算（边做边改）
- API简单
- 与高校合作推广（学生→企业→推动行业）
- 底层是PyTorch：百度PP飞桨等
