## 知识点清单

### 核心概念
1. view - 修改张量形状（只能处理连续张量）
2. contiguous - 将不连续张量转为连续
3. iscontiguous - 判断张量是否连续
4. 张量连续性：内存存储顺序 vs 逻辑顺序

### API/函数
| 函数名 | 用途 | 重要程度 |
|--------|------|----------|
| tensor.view(*shape) | 修改连续张量形状 | ⭐⭐⭐ |
| tensor.contiguous() | 转为连续张量 | ⭐⭐⭐ |
| tensor.iscontiguous() | 判断是否连续 | ⭐⭐ |

### 代码片段

```python
import torch

# 定义张量 (2, 3, 4)
t1 = torch.randn(2, 3, 4)

# 判断连续性
print(t1.iscontiguous())  # True

# 使用view修改形状
t2 = t1.view(24)  # → (24,)
t3 = t1.view(6, 4)  # → (6, 4)

# 经过transpose后变为不连续
t4 = t1.transpose(0, 1)  # → (3, 2, 4)
print(t4.iscontiguous())  # False

# view无法处理不连续张量，会报错
# t4.view(24)  # RuntimeError

# 先转为连续，再用view
t5 = t4.contiguous()  # 转为连续
t6 = t5.view(24)  # 成功
```
