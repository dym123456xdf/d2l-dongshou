## 知识点清单

### 核心概念
1. reshape - 改变形状不改变内容
2. unsqueeze - 在指定维度增加一个维度为1的维度（升维）
3. squeeze - 删除所有维度为1的维度（降维）
4. 维度越界问题

### API/函数
| 函数名 | 用途 | 重要程度 |
|--------|------|----------|
| tensor.unsqueeze(dim) | 在指定位置增加维度 | ⭐⭐⭐ |
| tensor.squeeze() | 删除所有为1的维度 | ⭐⭐⭐ |
| tensor.reshape(shape) | 改变张量形状 | ⭐⭐⭐ |

### 代码片段

```python
import torch

# 定义张量 (2, 3)
t1 = torch.tensor([[6, 9, 9], [2, 8, 7]])
print(t1.shape)  # torch.Size([2, 3])

# unsqueeze - 升维
t2 = t1.unsqueeze(0)  # 在0维增加 → (1, 2, 3)
t3 = t1.unsqueeze(1)  # 在1维增加 → (2, 1, 3)
t4 = t1.unsqueeze(2)  # 在2维增加 → (2, 3, 1)

# squeeze - 降维
t_squeezed = t2.squeeze()  # 删除所有为1的维度 → (2, 3)

# reshape 示例
t_reshaped = t1.reshape(3, 2)  # 变成 (3, 2)
# 数据拼接顺序不变：6,9,9,2,8,7 → 6,9/9,2/8,7
```
