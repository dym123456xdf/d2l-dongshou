## 知识点清单

### 核心概念
1. transpose - 交换两个维度
2. permute - 任意维度交换
3. 张量维度交换的本质
4. 返回新张量，不改变原张量

### API/函数
| 函数名 | 用途 | 重要程度 |
|--------|------|----------|
| tensor.transpose(dim0, dim1) | 交换两个维度 | ⭐⭐⭐ |
| tensor.permute(*dims) | 任意维度交换 | ⭐⭐⭐ |

### 代码片段

```python
import torch

# 定义张量 (2, 3, 4)
t1 = torch.randn(2, 3, 4)

# transpose - 只能交换两个维度
t2 = t1.transpose(0, 1)  # 交换0和1维 → (3, 2, 4)
t3 = t1.transpose(0, 2)  # 交换0和2维 → (4, 3, 2)

# permute - 可以一次性交换多个维度
t4 = t1.permute(2, 1, 0)  # → (4, 3, 2)
t5 = t1.permute(1, 0, 2)  # → (3, 2, 4)

# 注意：返回新张量，不改变原张量
print(t1.shape)  # 仍是 (2, 3, 4)
```
