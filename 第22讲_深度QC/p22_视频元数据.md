## 知识点清单

### 核心概念
1. reshape - 改变张量维度
2. squeeze - 降维（删除形状唯一的维度）
3. unsqueeze - 升维（增加形状唯一的维度）
4. transpose - 交换两个维度
5. permute - 任意维度交换
6. view - 修改连续张量形状
7. contiguous - 转为连续张量
8. iscontiguous - 判断是否连续

### API/函数
| 函数名 | 用途 | 重要程度 |
|--------|------|----------|
| tensor.reshape() | 改变张量形状 | ⭐⭐⭐ |
| tensor.squeeze() | 删除维度为1的维度 | ⭐⭐⭐ |
| tensor.unsqueeze(dim) | 在指定位置增加维度 | ⭐⭐⭐ |
| tensor.transpose(dim0, dim1) | 交换两个维度 | ⭐⭐⭐ |
| tensor.permute(*dims) | 任意维度交换 | ⭐⭐⭐ |
| tensor.view(shape) | 修改连续张量形状 | ⭐⭐ |
| tensor.contiguous() | 转为连续张量 | ⭐⭐ |
| tensor.iscontiguous() | 判断是否连续 | ⭐⭐ |

### 代码片段

```python
import torch

# 定义张量
t = torch.randn(2, 3, 4)  # 形状 (2, 3, 4)

# reshape
t_reshaped = t.reshape(6, 4)  # 变成 (6, 4)

# unsqueeze - 升维
t_unsqueezed = t.unsqueeze(0)  # 变成 (1, 2, 3, 4)

# squeeze - 降维
t_squeezed = t_unsqueezed.squeeze()  # 变回 (2, 3, 4)

# transpose - 交换维度
t_transposed = t.transpose(0, 1)  # 变成 (3, 2, 4)

# permute - 任意维度交换
t_permuted = t.permute(2, 1, 0)  # 变成 (4, 3, 2)

# view - 修改连续张量形状
t_view = t.view(24)  # 变成 (24,)

# 判断连续性
is_cont = t.iscontiguous()

# 转为连续
t_cont = t.contiguous()
```
