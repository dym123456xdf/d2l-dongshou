## 知识点清单

### 核心概念
1. torch.cat - 不改变维度数的拼接
2. torch.stack - 改变维度数的拼接（新增维度）
3. 维度匹配规则
4. 维度越界问题

### API/函数
| 函数名 | 用途 | 重要程度 |
|--------|------|----------|
| torch.cat(tensors, dim) | 按指定维度拼接张量，不改变维度数 | ⭐⭐⭐ |
| torch.stack(tensors, dim) | 按指定维度拼接张量，增加新维度 | ⭐⭐⭐ |

### 代码片段

```python
import torch

# 创建两个张量 (2, 3)
t1 = torch.randint(1, 10, (2, 3))
t2 = torch.randint(1, 10, (2, 3))

# cat - 不改变维度数
# 按dim=0拼接：(2,3)+(2,3) → (4,3)
t3 = torch.cat([t1, t2], dim=0)

# 按dim=1拼接：(2,3)+(2,3) → (2,6)
t4 = torch.cat([t1, t2], dim=1)

# stack - 改变维度数（新增维度）
# 按dim=0拼接：(2,3)+(2,3) → (2,2,3)
t5 = torch.stack([t1, t2], dim=0)

# 按dim=1拼接：(2,3)+(2,3) → (2,2,3)
t6 = torch.stack([t1, t2], dim=1)
```
