## 内容结构分析

### 一、课程回顾（上午内容）
1. **预算（运算）**
   - ADD：加法
   - ADD_（下划线）：修改原数据（in-place）
   - SUBTRACT：减法
   - MULTIPLY：乘法
   - DIVIDE：除法
   - NEG（取反数）

2. **函数**
   - 求和：sum
   - 求均值：mean
   - 求最大值：max
   - 求最小值：min
   - N次幂：power
   - 开平方：sqrt
   - E的N次方：exp
   - 对数：log, log2, log10

3. **索引**
   - 简单索引：T[行, 列]
   - 下标索引：0 vs :0（前者是第一行，后者是所有行的第一列）
   - 列表索引：中括号分组
   - 范围取值：切片操作
   - 布尔索引：条件判断

### 二、张量运算深入

#### 基本运算
```
ADD / SUBTRACT / MULTIPLY / DIVIDE
带下划线版本：修改原数据
```

#### 点乘
```
T1 * T2
T1.mul(T2)
```

#### 矩阵相乘
```
T1 @ T2
T1.matmul(T2)
T1.matrix_multiply(T2)
```
**注意**：A的列等于B的行，结果是A行B列

### 三、下午内容预告
- 形状拼接
- 自动微分
- 难度系数：运算难度1，索引难度2，微分难度3
