## 知识点清单

### 核心概念
1. 梯度累加问题 - 梯度不清零会累加
2. 梯度清零 - W.grad.zero_()
3. 非空判断 - if W.grad is not None
4. 循环迭代 - 逐步逼近最优解

### API/函数
| 函数名 | 用途 | 重要程度 |
|--------|------|----------|
| loss.backward() | 反向传播 | ⭐⭐⭐ |
| W.grad.zero_() | 梯度清零 | ⭐⭐⭐ |
| W.grad is not None | 非空判断 | ⭐⭐ |
| W.data | 获取参数值 | ⭐⭐⭐ |

### 代码片段

```python
import torch

# 1. 定义初始权重
W = torch.tensor(10.0, requires_grad=True, dtype=torch.float32)
learning_rate = 0.01

print(f"初始权重: {W}")  # 10.0
print(f"初始损失: {W**2 + 20}")  # 120

# 2. 循环迭代100次
for i in range(1, 101):
    # 正向传播：计算当前损失
    loss = W ** 2 + 20
    
    # 梯度清零（非空判断）
    if W.grad is not None:
        W.grad.zero_()
    
    # 反向传播：计算梯度
    loss.backward()
    
    # 更新权重
    W.data = W.data - learning_rate * W.grad

print(f"最终权重: {W.data}")  # 趋近最优解
print(f"最终损失: {loss.item()}")  # 从120降到21
```

| 1 | 9.8 | ~118 |
| 2 | 9.604 | ~112 |
| ... | ... | ... |
| 100 | 趋近0 | ~21 |
