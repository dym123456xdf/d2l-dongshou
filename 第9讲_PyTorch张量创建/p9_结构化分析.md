## 核心知识点

### 1. 张量定义
- **定义**: 存储同一类型元素的多维矩阵
- **特点**: 元素必须是数值
- **优势**: 可GPU加速、自动微分

### 2. 张量维度说明
| 维度 | 描述 | 举例 |
|------|------|------|
| 0维 | 标量(一个值) | tensor(30) |
| 1维 | 向量 | [1, 2, 3, 4] |
| 2维 | 矩阵(行列) | [[1,2],[3,4]] |
| 3维 | 多矩阵组合 | HWC/CHW格式图片 |
| 4维 | 批量数据 | 批次图片 |

### 3. 三种张量创建方式
```python
# 方式一：torch.tensor(data)
t1 = torch.tensor([1, 2, 3])

# 方式二：torch.Tensor(data)  
t2 = torch.Tensor([1, 2, 3])

# 方式三：torch.from_numpy(ndarray)
t3 = torch.from_numpy(np.array([1, 2, 3]))
```

### 4. 图像表示格式
- **HWC**: Height × Width × Channel
- **CHW**: Channel × Height × Width (PyTorch处理时)
