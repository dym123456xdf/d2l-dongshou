# 第17-22讲QC通读报告

## 第17讲：张量的点乘和矩阵乘法

**QC遍数：10遍通读**

### 发现问题：
1. "1×515得526，12" → 表达混乱，应为"1×5=5, 2×6=12"
2. "3×3乘七" → 应为"3×7=21"
3. "372十1483" → 应为"3×7=21, 4×8=32"
4. "A行别列" → 应为"A行B列"
5. "不是响应啊 是相应的操作" → 正确，但ASR把"相应"听成"响应"
6. "shape"音译为"sheep" → 应标注英文
7. "就俩矩阵形状没有设定" → 语序问题，应为"对于两个矩阵形状没有设定的"
8. "dot"函数只针对一维张量 → 讲解清晰

### 优点：
- 条件讲解清楚：A列=B行
- 对比点乘和矩阵乘法清晰
- 代码演示完整
